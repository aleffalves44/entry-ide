Style System · v1.2 proposal · Entry IDE

Eight themes,
one identity.

Today's Entry ships 30 themes, fragmenting the identity. This proposal cuts the catalog to eight: four dark, four light. Every theme inherits the same triple-font system, the same brass-rail accent, the same editorial layout grammar — they vary in mood, not in vocabulary. Plus a pass on the surfaces around them: status bar, empty states, onboarding, settings, notifications.
