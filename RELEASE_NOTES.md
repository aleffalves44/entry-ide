# Hermes IDE 1.3.1

A small follow-up on the subagents popover so it stops blending into
the page on some themes.

## Subagents popover stands out from the page

The little popover that opens when you click the **N subagents** chip
in the agent header now sits on a clearly distinct surface on every
theme. On the warm dark themes (Atelier, Observatory, Phosphor) the
popover had been painted only a hair lighter than the page behind it,
so it could read as part of the chat rather than as a floating panel.

The frosted themes (Frosted Dark, Frosted Light) now give the popover
the same Apple-style frosted-glass treatment that the rest of the
chrome already uses, so it no longer looks like the odd one out.

The text, layout, and behaviour of the popover are unchanged — only
the surface beneath them.
